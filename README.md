## 🤖 AI & Search Engine Optimization

This repository is configured for maximum visibility:

- ✅ **Schema.org validated** JSON-LD structured data
- ✅ **HTML landing pages** with embedded `<script type="application/ld+json">` tags
- ✅ **XML Sitemap** at `sitemap.xml` for search engine crawling
- ✅ **robots.txt** allowing all major AI crawlers (GPTBot, ClaudeBot, PerplexityBot, Google-Extended)
- ✅ **GitHub Actions** for automatic timestamp updates and monthly refreshes
- ✅ **Google & Bing ping** on monthly refresh to trigger re-indexing

## 🔄 Automated Maintenance

Two GitHub Actions workflows keep schemas fresh:

1. **Update Timestamps** — Runs on every push to `main`, updating `dateModified` fields
2. **Monthly Refresh** — Runs on the 1st of each month, refreshing all timestamps and pinging Google & Bing sitemaps
